# 🌍 3D World Explorer | Educational App

An interactive 3D web application designed to help students explore the world. Built with a focus on modern UI/UX principles, data visualization, and internationalization (i18n).

## Technologies Used
* **HTML5 & Vanilla JavaScript**: Core logic, DOM manipulation, and asynchronous data fetching (GeoJSON).
* **Three.js & Globe.gl**: 3D rendering and geographical data visualization with custom polygon extrusions and materials.
* **Tailwind CSS**: Rapid UI development using utility classes and responsive design.
* **Lucide Icons**: Lightweight and clean iconography.

## Features
* **Bilingual Support (i18n):** Seamlessly switch between English and Spanish.
* **Dual Exploration Modes:** Toggle between macro 'Continents' view and detailed 'Countries' view.
* **Interactive Data Visualization:** Custom hover effects, dynamic tooltip generation, and smooth altitude transitions.
* **Dynamic Information Panel:** Displays population, area, historical context, and key facts.

# 🌍 Explorador Mundial 3D | App Educativa

Una aplicación web interactiva en 3D diseñada para ayudar a los estudiantes a explorar el mundo. Construida con un enfoque en principios modernos de UI/UX, visualización de datos e internacionalización (i18n).

## Tecnologías Utilizadas
* **HTML5 y JavaScript (Vanilla)**: Lógica central, manipulación del DOM y peticiones asíncronas de datos (GeoJSON).
* **Three.js y Globe.gl**: Renderizado 3D y visualización de datos geográficos con extrusión de polígonos y materiales personalizados.
* **Tailwind CSS**: Desarrollo rápido de UI utilizando clases utilitarias y diseño responsivo.
* **Lucide Icons**: Iconografía ligera y limpia.

## Características
* **Soporte Bilingüe (i18n):** Cambio fluido entre Inglés y Español.
* **Modos de Exploración Dual:** Alterna entre una vista macro de 'Continentes' y una vista detallada de 'Países'.
* **Visualización de Datos Interactiva:** Efectos hover personalizados, tooltips dinámicos y transiciones suaves de altitud.
* **Panel de Información Dinámico:** Muestra población, área, contexto histórico y datos clave.
